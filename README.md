# WinFocus
Windows 聚焦壁纸 | Windows Focus Wallpaper

# 功能
 - 支持Windows锁屏出现的历史壁纸的预览与缓存
   - （系统会定期清理历史聚焦壁纸）
 - 多样的超清必应壁纸
 - 自定义桌面动态壁纸
 - 自定义Focus锁屏壁纸
 - ...
 
---

# 特点
- [x] 系统支持
  - [x] 支持Windows 11
  - [x] 支持Windows 10 （1803+）
- [x] 全新WinUI 3框架
- [x] 支持高分屏
  - [x] 2K
  - [x] 2.5K+
  - [x] 4K
- [x] 必应超清壁纸
  - [x] 2160P+（4K+）
  - [x] 1200P
  - [x] 1080P
  - [x] 720P
  - [x] 壁纸简介
- [x] 核心采用Win32 API实现
  - [x] 动态壁纸
  - [x] 桌面壁纸
- [x] More
