# WinFocus
Windows 聚焦壁纸 | Windows Focus Wallpaper

# 功能
 - 支持Windows锁屏出现的历史壁纸的预览与缓存
   - （系统会定期清理历史聚焦壁纸）
 - 多样的超清必应壁纸
 - 自定义桌面动态壁纸
 - ...

# 特点
- [x] 使用WinUI 3
- [x] 支持高分屏
- [x] 采用Win32 API实现
